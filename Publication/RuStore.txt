КАТЕГОРИЯ: Шопинг

ВОЗРАСТНАЯ КАТЕГОРИЯ: 7+

КРАТКОЕ ОПИСАНИЕ:
ShoeStore — удобный магазин обуви. Каталог, избранное, профиль, авторизация.

ОСНОВНОЕ ОПИСАНИЕ :
ShoeStore — мобильное приложение интернет-магазина обуви с современным интерфейсом.

ВОЗМОЖНОСТИ:
• Каталог товаров по категориям: Running, Tennis, Outdoor
• Добавление товаров в избранное
• Регистрация и вход в личный кабинет
• Восстановление пароля через email-код
• Редактирование профиля пользователя
• Просмотр истории заказов

ПРИЛОЖЕНИЕ РАЗРАБОТАНО:
• Jetpack Compose (Material Design 3)
• Kotlin 2.0.21
• Современная навигация Bottom Navigation

Готово к подключению серверного API
